###MiniDB是什么?
----
MiniDB是一个基于html5新特性**localStorage**的**本地**微型非关系型数据库,它封装了一些非常基本的数据库操作以应对某些需要小规模长期性存储需求

古人说"杀鸡焉用牛刀",希望MiniDB能做好一把锋利的杀鸡小刀~

文档请看[文档](https://github.com/david500834/Mini-DB/wiki)
